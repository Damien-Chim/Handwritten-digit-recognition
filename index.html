Handwritten digit recognition deep learning model

This is a deep learning model that uses artificial neural networks to process and predict human written digits

Do you know?

Recognising numbers is something that humans can do as early as the age of two, but do you know it's actually very difficult for computers to do so?

Hence, complicated algorithms such as activation functions, loss functions and back propagation needs to be implemented in order to "teach" computers to read numbers

Try it out!

Write any digits from 0-9 (one at a time) and press the Submit button. Your handwriting will be processed and the model will predict what you wrote

You can experiment with different handwritings, the model should still be able to analyse your handwriting

Some tips to obtain the best results:

  • Try to write as centred as possible in the canvas (very important)

  • Avoid writing near edges (also quite important)

  • Avoid writing too big or too small, moderate sized handwriting is recommended (you know it :) )

    • Disclaimer: this model is still under training hence it is not totally perfect, yet.
